# DSA Exam Study Roadmap (Corrected Syllabus)

Since we now have the exact topics for Units 1, 2, and 3, here is the corrected priority study sequence.

### Phase 1: High Yield & Easy Marks (2.5 Hours)
Start with topics that are mathematically systematic and easy to score full marks on.
*   **[Unit 1] Arrays & ADT (30 mins)**: Learn how to represent polynomials using arrays and how to write the algorithm for adding two polynomials. Memorize differences between Primitive and Non-primitive DS.
*   **[Unit 2] Queues (45 mins)**: Master **Circular Queues**. Know the enqueue/dequeue logic and how `front` and `rear` pointers move circularly `(rear = (rear + 1) % size)`.
*   **[Unit 2] Stacks (30 mins)**: Practice converting Infix expressions to Postfix/Prefix using the stack priority table. This is a guaranteed mechanical question.
*   **[Unit 4] Sorting (45 mins)**: Focus on the "Divide and Conquer" traces: **Merge Sort** and **Quick Sort**. Also skim Bubble Sort.

### Phase 2: Core Data Structures (3 Hours)
*   **[Unit 3] Linked Lists (1 Hour)**: Practice node insertions/deletions for **Doubly Linked List (DLL)**. Also understand how a Generalized Linked List (GLL) represents polynomials.
*   **[Unit 4] Hashing (45 mins)**: Know how to build a Hash Table with **Linear Probing** and **Separate Chaining**.
*   **[Unit 5] Basic Trees (1 Hour)**: Learn how to construct a binary tree given In-Order and Pre-Order traversals. Practice drawing Huffman Trees.
*   **[Unit 6] Graph Traversals (45 mins)**: Practice **BFS** (using Queue) and **DFS** (using Stack). 

### Phase 3: Advanced Topics (2 Hours)
*   **[Unit 5] Advanced Trees (1 Hour)**: **AVL Trees** (memorize LL, RR, LR, RL rotations) and **OBST** (Optimal Binary Search Tree) construction table.
*   **[Unit 6] Shortest Path & MST (1 Hour)**: Trace **Dijkstra's Algorithm**, **Prim's**, and **Kruskal's**. Know the **Floyd-Warshall** transitive closure matrices.

*Focus your energy on writing clean algorithms and showing your step-by-step trace tables!*
